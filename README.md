# iOS Concepts Using Apps

This repository contains small iOS apps that I built while learning iOS development.

The purpose of this repository is to practice different iOS concepts by building simple applications. Each app in this project focuses on understanding a specific concept in iOS development.

While learning Swift and iOS development, I created these apps to better understand how iOS applications work.

---

## What This Repository Contains

This repository includes small practice applications that demonstrate different iOS concepts such as:

• Basic iOS app development  
• UI design using Storyboard  
• Navigation between screens  
• Working with buttons and user interaction  
• Learning Swift programming concepts  
• Understanding how iOS apps are structured  

Each app is created mainly for learning and experimenting with new ideas.

---

## Technologies Used

• Swift  
• Xcode  
• iOS SDK  
• Storyboard for UI design  

---

## How to Run the Project

1. Install **Xcode** on your Mac.

2. Clone this repository.


git clone https://github.com/araWINd-AR/IOS_CONCEPTS_USING_APPS.git   


3. Open the project folder.

4. Open the `.xcodeproj` file in Xcode.

5. Click **Run ▶** to build and run the application in the simulator.
